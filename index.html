Post-Covid Test

While we know that 2020 was tough, all that time spent streaming had to have been good for something, right? The Post-Covid Test will make sure that time was well spent.

You will be presented with several images from popular Netflix shows for which you have to guess the correct show AND the correct actor. Each correct answer will be worth one point so if you get both the actor and show correct it'll be two on the board for ya. Good Luck!
